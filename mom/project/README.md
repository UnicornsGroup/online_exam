# 📋 Single-Page Gujarati Progressive Form Wizard & Production Inventory PDF Generator

A lightweight client-side application designed to reproduce the official structured document layout of the uploaded reference **"ગોળ મસાલાનું વપરાશ પત્રક"** tracking document.

## 🚀 Key Functional Integrations

1. **Multi-Column Matrix Processor:** The wizard captures complicated multi-row tracking fields using an isolated interactive grid block mapped to dynamic state engines cleanly.
2. **Precision Canvas Renderer:** Combines `html2canvas` layout calculations with `jsPDF` coordinate tracking maps to reproduce clear borders, font sizes, and header grids in landscape mode.
3. **Automated Intermittent Storing:** Keeps form responses persistent within browser session tracks (`LocalStorage`) automatically to prevent loss during unexpected device closures.
